 ================================== Overview ==================================
This program implements a fast algorithm to solve the knapsack problem.

=============================== Run The Program ===============================
 1) Install Visual studio.
 2) Clone the GitHub repository. 
 3) Inputs are hardcoded as arrays in the source code, change these if you like.
 4) Run the program.
 5) Outputs are printed to console.
